# AI-Powered Server Log Management System

## Aim

Develop an intelligent log monitoring system that analyzes server logs and detects anomalous activities using machine learning techniques, helping identify potential errors and suspicious behavior.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Flask
* Matplotlib
* Joblib

## Key Features

* Parses and processes server log files
* Performs feature engineering on log data
* Detects anomalies using a machine learning model (Isolation Forest)
* Highlights suspicious log entries automatically
* Provides an interactive Flask-based web dashboard
* Displays summary statistics such as total logs, anomalies, and normal logs
* Exports results to CSV files for further analysis

## Output Files

* `anomalies.csv` – Contains detected anomalous log entries
* `predictions.csv` – Contains prediction results generated by the model
* Web Dashboard – Displays processed logs and anomaly summaries in a user-friendly interface

## How to Run

1. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
2. Start the application:

   ```
   python main.py
   ```
3. Open your browser and visit:

   ```
   http://127.0.0.1:5000
   ```

## Project Outcome

The system successfully automates server log analysis by identifying unusual patterns and presenting them through a simple dashboard, enabling faster monitoring and troubleshooting.
